# 🛍️ Customer Shopping Analysis Project

## 📌 Overview

This project focuses on analyzing customer shopping behavior to generate actionable business insights. The analysis covers data exploration, cleaning, SQL-based querying, and visualization through an interactive dashboard.

The goal is to identify purchasing patterns, customer preferences, and key trends that can support data-driven decision-making.

---

## 📂 Dataset

* The dataset contains customer transaction records, including:

  * Customer details
  * Product/category information
  * Purchase history
  * Review ratings
  * Discount usage

* Data is loaded and processed using Python before being stored in PostgreSQL for further analysis.

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy) – Data loading, cleaning, and EDA
* **PostgreSQL** – Data storage and SQL analysis
* **SQL** – Aggregations, joins, ranking, and business queries
* **Power BI** – Dashboard creation and visualization

---

## 🔄 Project Steps

### 1. Data Loading

* Imported dataset into Python using Pandas
* Inspected structure, data types, and missing values

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions (ratings, purchases, categories)
* Identified trends and anomalies
* Explored relationships between variables

### 3. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized column formats
* Ensured data consistency for analysis

### 4. SQL Analysis (PostgreSQL)

* Loaded cleaned data into PostgreSQL
* Performed queries such as:

  * Top-selling products
  * Highest-rated items
  * Category-wise performance
  * Discount usage trends
  * Customer behavior insights

### 5. Data Visualization (Power BI)

* Built an interactive dashboard to present insights
* Enabled filtering by category, product, and customer metrics

---

## 📊 Dashboard Features

* Sales and purchase trends
* Top-performing products and categories
* Average review ratings
* Discount usage analysis
* Customer behavior insights

---

## 📈 Key Results

* Identified top products based on purchase frequency and ratings
* Highlighted categories with highest customer engagement
* Analyzed impact of discounts on purchasing behavior
* Provided insights to improve marketing and sales strategies

## ▶️ How to Run the Project

1. **Clone the repository**

   ```bash
   git clone <repository-link>
   ```

2. **Install required Python libraries**

   ```bash
   pip install pandas numpy psycopg2
   ```

3. **Run Python scripts**

   * Load and clean the dataset
   * Perform EDA

4. **Set up PostgreSQL**

   * Create a database
   * Import the cleaned dataset
   * Run SQL queries provided in the project

5. **Open Power BI Dashboard**

   * Load the dataset or connect to PostgreSQL
   * Open the `.pbix` file to view the dashboard
